RAM minne eller Random Access Memory är ett minne där varje minnescell kan läsas eller skrivas till direkt utan att behöva läsa igenom andra delar av minnet tillskillnad från minnen som läses sekventiellt
RAM finns i datorn eftersom att vissa delar i datorn har väldigt lite eller iget minne alls och då kan de ta utav RAM och utföra sin process
